# CurrentMoment

`CurrentMoment` is a UIKit-based private social moments app built as a production-like interview project.

## What It Shows

- UIKit-first architecture
- `MVVM + Coordinator + DI`
- `AVFoundation` camera flow
- Firebase-ready repository layer
- Realtime history / friends flow
- Clean modular structure
- Free-account friendly setup
